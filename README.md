

# 🌍 Travel Economy Idea Hub

🚀 A modern full-stack web application designed to simplify travel planning, explore cost-effective ideas, and enhance user experience with intelligent features like chatbot assistance and data-driven insights.

---

## 📌 Overview

**Travel Economy Idea Hub** is a platform that helps users:

* Discover **budget-friendly travel ideas**
* Explore destinations efficiently
* Interact with an **AI-powered chatbot**
* Manage travel-related data seamlessly

The goal is to provide a **smart, user-friendly travel planning experience** in one place.

---

## ✨ Features

* 🔍 Smart travel idea exploration
* 🤖 AI Chatbot integration (Gemini API)
* 📊 Data-driven insights for travel planning
* 🔐 Secure user authentication
* ⚡ Fast and responsive UI
* 🌐 Full-stack architecture

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/alaharilakshyan/Travel-Economy-Idea-Hub.git
cd Travel-Economy-Idea-Hub
```

---

### 2️⃣ Setup Backend

```bash
cd backend
npm install
npm start
```

---

### 3️⃣ Setup Frontend

```bash
cd frontend
npm install
npm start
```

---

## 🔑 Environment Variables

Create a `.env` file in the backend folder and add:

```env
PORT=5000
MONGO_URI=your_mongodb_connection
GEMINI_API_KEY=your_api_key
```
